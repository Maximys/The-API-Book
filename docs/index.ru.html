Разработка API — особый навык: API является как мультипликатором ваших возможностей, так и мультипликатором ваших ошибок. Эта книга написана для того, чтобы поделиться опытом и изложить лучшие практики проектирования и управления продуктом API.

Вы можете скачать книгу «API» в формате PDF или EPUB, или читать её онлайн.
